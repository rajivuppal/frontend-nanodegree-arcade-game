Frogger Arcade Game:

1) To launch the game open the index.html file in the browser.
2) The game has a player in bottom center of the Game bard as well as a few enemies 
that are constantly moving across the board.
3) The goal of the game is for the player to be able to move up the game board and get to the water at the top.
4) Once the player reaches the top water portion, that is considered a success.
5) In order to get across to the top, the player has to avoid collisions with the enemy bugs that are moving across 
the board at various speeds.
6) The player can move up, down, left or right by pressing on the corresponding arrow keys on the keyboard.
7) Anytime the player collides with any one of the enemies, the player's position will be reset to the starting position.
